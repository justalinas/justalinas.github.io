Info: An Interactive English Project: The Scottsboro Trials

This is a special-made assignment that my partner, Jumhelle Viduya, and I made for an assignment we had in English about the Scottsboro boys.

With the code we have created, you will be able to easily see all of our information that we included. This includes the diary entries we wrote about some of Scottsboro people, more information discussing where you can learn more about them, and our info.

Even though we have the same style of coding, some of the information will be different. This is to fit what we did for the assignment / to get you to know us more.

We have also created a flow chart to help us design and organize how our coding was laid out.
